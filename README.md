# NimbusArc Cloud

Visual cloud architecture learning platform. The first active track is AWS SAA.

## Phase 0 Scope

- React + TypeScript app shell
- Separated `ui`, `features`, `domain`, `data`, and `integrations` folders
- Provider-aware AWS seed data
- Design tokens and first layout shell
- Project workflow docs for standards, task tracking, and visual QA
